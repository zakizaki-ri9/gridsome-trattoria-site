# Web調査レポート: 個人飲食店におけるメニュー表示と外部サイト（食べログ等）への誘導UI

## 調査メタデータ
- **調査日**: 2026-03-28
- **トピック**: 個人経営飲食店（イタリアン、フレンチ、カフェなど）のウェブサイトにおける、メニュー画像の使われ方と外部グルメサイト（食べログ等）への誘導デザインの事例調査。
- **調査深度**: 浅く広く（MUUUUU.ORG等のギャラリーやGoogle検索経由で複数サイトを閲覧）
- **情報源の信頼性**: 一般公開されている実際の飲食店HP、デザインアワード掲載サイト

## 調査課題
自社サイト内でのメニュー画像運用負荷を減らすため、「掲載画像はあくまでイメージとし、最新・詳細なメニューは食べログへ誘導する」という要件に対し、一般的な個人飲食店がどのようなUI/UXを採用しているかを把握する。

## 概要と実践的な知見
数店舗の事例を調査した結果、メニューの完全な一覧をサイト内に掲載しているケースは少なく、多くが「店舗の雰囲気や世界観を伝えるシズル感（Sizzle）重視のギャラリー」と「実用的なメニュー詳細（PDFや食べログ連携）」を分離していることがわかった。

### 1. メニュー画像の使われ方について
- **代表的な料理のみの掲載**: メニュー全品ではなく、「シェフのおすすめ」や「代表的なコース」の写真を数枚見せるLP（ランディングページ）形式が主流。
- **イメージとしての強調**: 「メニュー内容は季節や仕入れにより変更になる場合がございます。掲載画像はイメージです」といった免責事項と共にギャラリーとして見せる構成が多い。

### 2. 最新メニュー（食べログ等）への誘導手法
- **専用のボタン配置**: 「Menu」セクションの最後に、目立つアクセントカラーで「最新のメニューはこちら（食べログ）」「ご予約・詳細はこちら」という外部ボタンを配置する手法が非常に普及している。
- **PDFリンクの実装**: 日替わりや月替りのメニューを自社運用する場合は、PDFのリンクを一部だけ差し替える運用をしている中小規模の店舗（例: Cantina Zushiなど）も見受けられた。
- **SNS連携**: InstagramのFeedをそのまま埋め込み、Today's Specialをそちらに委ねるカフェ・バル業態も主流。

### 3. レイアウト構造
1. **セクション見出し**（例: LUNCH / DINNER）
2. **概算価格帯のテキスト**（例: Dinner Course ¥4,000〜）
3. **シズル感のあるイメージ画像グリッド**（3〜6枚程度）
4. **外部導線ボタン**

## 推奨事項
Trattoria Portoのサイトにおいても、現在のギャラリー画像を「イメージ」として活用しつつ、構成を以下のようにアップデートすることを推奨する。
1. 現在の `Menu` ページに「当店のメニュー（一例）です。価格や内容は変動する場合がございます。最新のメニューは食べログ、Instagram、または当店にてご確認ください」という説明テキストを追加。
2. ギャラリー（MenuGalleryコンポーネント）の上部または下部に、「食べログで確認する」「Instagramで確認する」という大きなボタン（CTA）を配置する。
